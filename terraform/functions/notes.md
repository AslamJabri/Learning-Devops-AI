# Basic of Function
A function is a block of code that performs a specific task.
Terraform has wide variety of functions available to achieve different set of use-cases.
Functions are grouped into categories.Some of these include:
 Function Categories            Function Available
 Numeric Function               abs,ceil,floor,max,min
 String Function                concat,replace,split,tolower,toupper
 Collection Functions           elements,keys,length,merge,sort
 FileSystem Function            file,filebase64,dirname

 TERRAFORM LANG  does not support user-defined functions and so only the function build in to the lang are available to use.
